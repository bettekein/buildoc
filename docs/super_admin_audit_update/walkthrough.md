# ウォークスルー: スーパー管理者機能と監査ログ・マスタ機能強化

## 1. スーパー管理者機能
- `user@buildoc.test` に `Super Admin` ロールを付与しました。
- ナビゲーションバーに「システム管理」メニューを追加し、以下の機能へアクセス可能にしました。
    - **テナント管理**: テナントの作成・編集・削除・一覧表示が可能。
    - **監査ログ**: `laravel-auditing` を使用し、データの変更履歴（誰が、いつ、何を）を記録・閲覧可能。検索やフィルター機能も実装。

## 2. データベース監査と論理削除
- 全テーブル (`users`, `tenants`, `customers`, `projects`, `staff`, `vehicles`, `tools` 等) に以下のカラムを追加しました。
    - `deleted_at`:論理削除用 (SoftDeletes)
    - `created_by`: 作成者ID (Userstamps)
    - `updated_by`: 更新者ID (Userstamps)
    - `deleted_by`: 削除者ID (Userstamps)
- これにより、データの削除は物理削除ではなく論理削除となり、後から復元可能になりました。また、操作者の追跡が可能になりました。

## 3. マスタ管理の機能強化
各マスタ管理画面（スタッフ、車両、工具、顧客）に対し、以下の機能を追加中です（スタッフマスタは完了）。
- **検索機能**: 名前などでレコードを絞り込み可能。
- **ゴミ箱機能**: 削除されたレコードの表示、復元、完全削除が可能。
